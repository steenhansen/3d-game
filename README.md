## Frames Per Second
    Galaxy S24 FE (2024)
        Chrome 60 to 80 fps
        Samsung 60 fps
        Firefox 45 fps

    Galaxy SM-A520 (2017) 
        Chrome 6 fps - when degraded
        Firefox 3 fps - when degraded
        Samsung does not work

    AMD Ryzen 7 3700X (2019)
        Chrome 60 fps
        Opera 60 fps
        Brave 60 fps
        Edge 60 fps
        Firefox 50 to 60 - when degraded
